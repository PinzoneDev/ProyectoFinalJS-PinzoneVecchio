La aplicación "Simulador de Cambio de Divisas" es una herramienta diseñada para calcular conversiones de divisas de manera sencilla y rápida. Permite a los usuarios ingresar un monto en una divisa, seleccionar tanto la divisa de origen como la de destino, y obtener el monto convertido de una divisa a otra.

Esta aplicación se creó utilizando una combinación de tecnologías y conceptos. Entre ellos se incluyen:

HTML y CSS: Se utilizaron para estructurar y estilizar la interfaz de usuario, creando una experiencia amigable y atractiva.

JavaScript: Se empleó para la lógica de la aplicación, manejando las solicitudes a la API de tasas de cambio, procesando la conversión de divisas y actualizando dinámicamente la interfaz de usuario con los resultados.

Fetch/Axios: Se utilizó para realizar solicitudes a una API externa, permitiendo obtener las tasas de cambio en tiempo real entre diferentes divisas.

Eventos y Manipulación del DOM: Los eventos del usuario, como clics y selecciones, se capturaron para manejar las acciones de conversión. Se manipuló el DOM para reflejar los cambios y mostrar los resultados de manera dinámica en la interfaz.

Bootstrap: Se implementó para mejorar la apariencia y la disposición de los elementos, permitiendo un diseño responsivo y una estructura más organizada.

El objetivo principal de la aplicación es proporcionar una solución interactiva y útil para usuarios que requieran calcular conversiones de divisas de forma práctica y eficiente.
